Code Coverage report in Sonarqube with Jacoco

Steps
  • Configure Maven with Jacoco. Please refer this link
  • Setup Sonarqube with Sonar scanner. Please refer this link
  • Configure Sonar scanner to refer to the project to generate code coverage report
  • Run sonarqube
  • Run sonar scanner
  • Open report in the browser
Configure Sonar scanner to refer to the project to generate code coverage report
  • I am configuring maven project
  • Go to unzipped sonar-scanner-3.0.1.733/conf, open sonar-scanner.properties using any editor
  • Give the following properties to configure the project: 
    
sonar.projectKey=MavenJacocoIntegration
sonar.projectName=MavenJacoco
sonar.projectVersion=1.0
sonar.language=java
sonar.projectBaseDir=E:/Backup/JavaPrep/practiceProjects/Maven/MavenJacocoIntegration
sonar.sources=src/main/java
sonar.java.binaries=target/classes
sonar.tests=src/test/java
sonar.java.test.binaries=target/test-classes
sonar.junit.reportsPath=target/surefire-reports
sonar.jacoco.reportPaths=target/coverage-reports/jacoco-ut.exec
sonar.verbose=true
Run sonarqube
Run sonar scanner
  • Starting the sonar scanner
  • Sonar scanner completed and generated the report at http://localhost:9000/dashboard/index/MavenJacocoIntegration
Open report in the browser
  • Open http://localhost:9000/dashboard/index/MavenJacocoIntegration URL in the browser
