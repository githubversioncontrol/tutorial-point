Create Java Project using Maven and Import into Eclipse

Steps
  • Install Maven : refer Install Maven tutorial
  • Create Core Java Project using Maven
  • Import that project into Eclipse
  • Update compiler version of project
Create Core Java Project using Maven
  • Open Command Prompt
  • Navigate to folder where you want to create java project
  • Run following command

mvn archetype:generate -DgroupId=com.maven.practice -DartifactId=MavenHelloWorld -Dversion=1.0-SNAPSHOT -Dpackage=maven.hello.world -DarchetypeArtifactId=maven-archetype-quickstart -DinteractiveMode=false
Import project into Eclipse
  • Go to the location where project is created
  • Import projec to Eclipse
  • Open Eclipse -> Right click in Project Explorer -> Import -> Import -> Maven -> Existing Maven Projects -> Next -> Browse to Maven project location -> Finish
