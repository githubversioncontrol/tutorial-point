Create React Application from cmd
1. Install create-react-app using npm if not installed already
cmd> npm install -g create-react-app
2. Create react-bootstrap. This will take some time.
cmd> create-react-app react-bootstrap
3. Go inside app created above
cmd>cd react-bootstrap
4. Install react and react-dom and save to package.json
..\react-bootstrapd>npm install --save react react-dom
5. Install bootstrap
..\react-bootstrapd>npm install --save bootstrap
6. Open project folder in any editor (visual-code, sublime etc)
7. Add import '../node_modules/bootstrap/dist/css/bootstrap.min.css'; entry to index.js
