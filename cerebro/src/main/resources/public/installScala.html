Install Scala
  • Download Scala from Here
  • This will download scala-2.12.1.msi
  • Double click scala-2.12..1.msi and follow instructions in instllation wizard
  • Once installation completed, open command prompt and run scala -version
  • Enter and Exit Scala
Install sbt (Simple Build Tool)
  • Download sbt from Here
  • This will download sbt-0.13.13.1.msi
  • Double click sbt-0.13.13.1.msi and follow instructions in installation wizard
  • Once installation completed, open command prompt and run sbt
  • Exit sbt
Install sbt (Simple Build Tool)
